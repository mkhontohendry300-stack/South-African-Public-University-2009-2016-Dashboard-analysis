##  Project Overview

This project analyses longitudinal data from South African public universities spanning **2009 to 2016**, transforming raw institutional records into strategic, decision-ready insights across five critical domains:

| Domain | Focus |
|---|---|
|  **Student Enrollments** | Trends by gender, race, and field of study |
|  **Graduation Rates** | Success rates and throughput analysis |
|  **Faculty Demographics** | Staff composition and student-to-staff ratios |
|  **Financial Performance** | Income, expenditure, and budget efficiency |
|  **Research Outputs** | Publication trends and funding efficiency |

---

##  Objectives

- ✅ Clean and prepare large institutional datasets using **Excel**
- ✅ Design and query relational databases using **SQL (PostgreSQL)**
- ✅ Build interactive executive dashboards in **Power BI**
- ✅ Extract multi-year insights on enrollment, graduation, faculty, finances, and research
- ✅ Present findings in structured, professional analytical reports

##  Key Analysis & Deliverables

### 1.  Data Cleaning & Preparation — *Excel, Python*

- Standardised column names and data formats across all datasets
- Converted all date fields to `YYYY-MM-DD` format
- Handled missing values and removed duplicate records
- Applied conditional formatting to surface anomalies

> **Deliverable:** Cleaned, validated dataset ready for SQL ingestion.

---

### 2.  SQL Database & Analysis — *PostgreSQL*

Designed a normalised relational schema and calculated the following metrics:

- Enrollment trends over time and top-performing universities by headcount
- Graduation success rates by year, field, gender, and race
- Student-to-staff ratios and faculty demographic breakdowns
- Research output volumes vs. funding received
- Financial income and expenditure patterns over 8 years

> **Deliverable:** Optimised SQL query library and exported analysis results.

---

### 3.  Power BI Dashboards

Four fully interactive dashboards built for executive-level decision-making:

| Dashboard | Key Visuals |
|---|---|
|  **Enrollment & Graduation** | Multi-year trends, comparisons, filters by Gender, Race, Field of Study |
|  **Faculty Demographics** | Gender & race composition, student-to-staff ratio heatmaps |
|  **Financial Overview** | Income vs. expenditure analysis, budget trend lines |
|  **Research Performance** | Output heatmaps, funding efficiency ratios, KPI indicators |

> **Deliverable:** Interactive dashboards ready for stakeholder presentation and strategic use.
